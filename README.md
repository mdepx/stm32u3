# look

Brushless motor driving app using NUCLEO-U385RG-Q

### Set up compiler
    $ sudo apt install gcc-arm-none-eabi

### Get sources and build the project
    $ git clone --recursive https://github.com/mdepx/look
    $ cd look
    $ make clean all
