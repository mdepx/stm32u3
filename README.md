# stm32u3

Brushless motor driving app using NUCLEO-U385RG-Q and EVLDRIVE102BH.

### Set up compiler
    $ sudo apt install gcc-arm-none-eabi

### Get sources and build the project
    $ git clone --recursive https://github.com/mdepx/stm32u3
    $ cd stm32u3
    $ make clean all
